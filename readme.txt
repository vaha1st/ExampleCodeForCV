Цель: упражнение кодинга и оформления кода. Здесь я не пытаюсь сделать легковесную программу, четко выполняющую свою задачу, а применяю новые полученные знания, в рамках функциональности конвертера и закрепляю эти знания. Поэтому множество решений может показаться абсурдными, например хранение истории конвертаций в SQL, когда это можно сделать, допустим, по средствам коллекций. Я просто беру новую технологию и пытаюсь ее внедрить в этот проект.

Возможности:
- Простая конвертация по значениям;
- Конвертация вводимых значений с консоли;
- Сохранение истории конвертаций в файл или в SQL. Вызов/очистка истории;
- Web форма для простой конвертации;
- Использованные технологии: Java 13, Spring Core & MVC, Hibernate Validator, Oracle JDBC.
